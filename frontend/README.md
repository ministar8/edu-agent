# Frontend

Next.js application for the student workspace, multi-agent chat, practice workflow, knowledge map, and management/debug views.

## Key paths

- `src/app/`: Next.js App Router entry points.
- `src/features/`: feature modules for chat, practice, knowledge map, agent flow, and admin debug panels.
- `src/shared/`: shared UI, API, auth, and utility code.

## Local commands

```bash
npm install
npm run dev
npm run lint
npm run typecheck
```
