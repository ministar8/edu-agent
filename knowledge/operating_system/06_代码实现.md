# 操作系统代码实现

## ❗【考点】银行家算法

银行家算法是死锁避免算法，在进程申请资源时，先判断分配后系统是否仍处于安全状态。

### 数据结构

```c
int Available[m];     // 每类资源可用数量
int Max[n][m];        // 每个进程的最大需求
int Allocation[n][m]; // 每个进程已分配数量
int Need[n][m];       // 每个进程还需数量 = Max - Allocation
```

### 安全性检查算法

```c
int IsSafe(int Available[], int Max[][M], int Allocation[][M], int Need[][M], int n, int m) {
    int Work[M], Finish[N];
    // Step 1: 初始化
    for (int j = 0; j < m; j++) Work[j] = Available[j];
    for (int i = 0; i < n; i++) Finish[i] = 0;
    
    // Step 2: 找满足条件的进程
    int found;
    do {
        found = 0;
        for (int i = 0; i < n; i++) {
            if (!Finish[i]) {
                int can_allocate = 1;
                for (int j = 0; j < m; j++) {
                    if (Need[i][j] > Work[j]) { can_allocate = 0; break; }
                }
                if (can_allocate) {
                    // Step 3: 模拟分配并释放
                    for (int j = 0; j < m; j++) Work[j] += Allocation[i][j];
                    Finish[i] = 1;
                    found = 1;
                }
            }
        }
    } while (found);
    
    // Step 4: 检查所有进程是否完成
    for (int i = 0; i < n; i++)
        if (!Finish[i]) return 0;  // 不安全
    return 1;  // 安全
}
```

### 资源请求算法

```c
int RequestResources(int pid, int request[], ...) {
    // Step 1: 检查请求是否超过 Need
    for (int j = 0; j < m; j++)
        if (request[j] > Need[pid][j]) return ERROR;
    
    // Step 2: 检查可用资源是否足够
    for (int j = 0; j < m; j++)
        if (request[j] > Available[j]) return WAIT;
    
    // Step 3: 试探性分配
    for (int j = 0; j < m; j++) {
        Available[j] -= request[j];
        Allocation[pid][j] += request[j];
        Need[pid][j] -= request[j];
    }
    
    // Step 4: 安全性检查
    if (IsSafe(...)) return GRANT;  // 安全，正式分配
    
    // 不安全，回滚
    for (int j = 0; j < m; j++) {
        Available[j] += request[j];
        Allocation[pid][j] -= request[j];
        Need[pid][j] += request[j];
    }
    return WAIT;
}
```

## ❗【考点】信号量实现生产者-消费者问题

生产者-消费者问题是最经典的进程同步问题。缓冲区大小为 n，需要三个信号量。

### 信号量定义

```c
semaphore mutex = 1;   // 互斥访问缓冲区
semaphore empty = n;   // 空缓冲区数量（初始为 n）
semaphore full  = 0;   // 满缓冲区数量（初始为 0）
```

### 生产者进程

```c
void Producer() {
    while (1) {
        Item item = produce_item();  // 生产产品
        P(empty);    // 申请空缓冲区（必须先于 P(mutex)）
        P(mutex);    // 进入临界区
        put_item(item);  // 放入产品
        V(mutex);    // 离开临界区
        V(full);     // 增加满缓冲区计数
    }
}
```

### 消费者进程

```c
void Consumer() {
    while (1) {
        P(full);     // 申请满缓冲区（必须先于 P(mutex)）
        P(mutex);    // 进入临界区
        Item item = get_item();  // 取出产品
        V(mutex);    // 离开临界区
        V(empty);    // 增加空缓冲区计数
        consume_item(item);  // 消费产品
    }
}
```

**注意**：P(empty/full) 必须在 P(mutex) 之前，否则可能死锁。如果先 P(mutex) 再 P(empty)，当缓冲区满时生产者持有 mutex 等待 empty，消费者需要 mutex 才能消费释放 empty，形成死锁。

## ❗【考点】读者-写者问题

### 读者优先

```c
semaphore rw = 1;     // 控制写者互斥
semaphore mutex = 1;  // 保护 read_count
int read_count = 0;

void Reader() {
    P(mutex);
    read_count++;
    if (read_count == 1) P(rw);  // 第一个读者加锁
    V(mutex);
    // 读操作...
    P(mutex);
    read_count--;
    if (read_count == 0) V(rw);  // 最后一个读者解锁
    V(mutex);
}

void Writer() {
    P(rw);
    // 写操作...
    V(rw);
}
```

### 写者优先（读写公平）

```c
semaphore rw = 1;
semaphore mutex = 1;
semaphore w = 1;      // 写优先信号量
int read_count = 0;

void Reader() {
    P(w);             // 在写者等待时阻止新读者
    P(mutex);
    read_count++;
    if (read_count == 1) P(rw);
    V(mutex);
    V(w);
    // 读操作...
    P(mutex);
    read_count--;
    if (read_count == 0) V(rw);
    V(mutex);
}

void Writer() {
    P(w);             // 阻止后续读者
    P(rw);
    // 写操作...
    V(rw);
    V(w);
}
```

## ❗【考点】哲学家进餐问题

五个哲学家围坐，每人左右各一根筷子，需要同时拿到两根才能进餐。

### 死锁避免方案：限制同时拿筷子的人数

```c
semaphore chopstick[5] = {1, 1, 1, 1, 1};
semaphore limit = 4;  // 最多4人同时拿筷子

void Philosopher(int i) {
    while (1) {
        think();
        P(limit);
        P(chopstick[i]);
        P(chopstick[(i + 1) % 5]);
        eat();
        V(chopstick[(i + 1) % 5]);
        V(chopstick[i]);
        V(limit);
        think();
    }
}
```

## ❗【考点】进程调度算法代码

### 时间片轮转（RR）

```c
void RoundRobin(Queue *ready_queue, int time_slice) {
    while (!IsEmpty(ready_queue)) {
        PCB *p = DeQueue(ready_queue);
        int run_time = min(time_slice, p->remaining_time);
        p->remaining_time -= run_time;
        if (p->remaining_time > 0) {
            EnQueue(ready_queue, p);  // 未完成，重新入队
        } else {
            // 进程完成
        }
    }
}
```
