# 数据结构代码实现

## 二叉树遍历

### ❗【考点】二叉树前序遍历（递归）

前序遍历顺序：根 → 左 → 右

```c
void PreOrder(BiTree T) {
    if (T != NULL) {
        visit(T);           // 访问根结点
        PreOrder(T->lchild); // 递归遍历左子树
        PreOrder(T->rchild); // 递归遍历右子树
    }
}
```

### ❗【考点】二叉树前序遍历（非递归，用栈）

非递归前序遍历用栈模拟递归过程：先访问根，再遍历左子树，最后遍历右子树。

```c
void PreOrderNonRecursive(BiTree T) {
    Stack S;
    InitStack(S);
    BiTree p = T;
    while (p || !IsEmpty(S)) {
        if (p) {
            visit(p);        // 访问根结点
            Push(S, p);      // 根结点入栈（为了后续访问右子树）
            p = p->lchild;   // 走向左子树
        } else {
            Pop(S, p);       // 弹出栈顶
            p = p->rchild;   // 走向右子树
        }
    }
}
```

### 二叉树后序遍历（递归）

后序遍历顺序：左 → 右 → 根

```c
void PostOrder(BiTree T) {
    if (T != NULL) {
```

### ❗【考点】二叉树后序遍历（非递归，用栈）

后序遍历非递归实现较复杂，需要记录最近访问的结点，防止右子树重复访问。

```c
void PostOrderNonRecursive(BiTree T) {
    Stack S;
    InitStack(S);
    BiTree p = T, r = NULL;
    while (p || !IsEmpty(S)) {
        if (p) {
            Push(S, p);
            p = p->lchild;
        } else {
            GetTop(S, p);
            if (p->rchild && p->rchild != r) {
                p = p->rchild;  // 右子树尚未访问
            } else {
                Pop(S, p);
                visit(p);
                r = p;          // 记录最近访问的结点
                p = NULL;
            }
        }
    }
}
```

### 二叉树层序遍历（用队列）

```c
void LevelOrder(BiTree T) {
    Queue Q;
    InitQueue(Q);
    BiTree p;
    EnQueue(Q, T);
    while (!IsEmpty(Q)) {
        DeQueue(Q, p);
        visit(p);
        if (p->lchild) EnQueue(Q, p->lchild);
        if (p->rchild) EnQueue(Q, p->rchild);
    }
}
```

## ❗【考点】线索二叉树

利用二叉链表的空指针域，指向结点在某种遍历序列中的前驱和后继。

```c
typedef struct ThreadNode {
    ElemType data;
    struct ThreadNode *lchild, *rchild;
    int ltag, rtag;  // 0: 孩子, 1: 线索
} ThreadNode, *ThreadTree;

// 中序线索化
void InThread(ThreadTree p, ThreadTree *pre) {
    if (p) {
        InThread(p->lchild, pre);
        if (!p->lchild) { p->ltag = 1; p->lchild = *pre; }
        if (*pre && !(*pre)->rchild) { (*pre)->rtag = 1; (*pre)->rchild = p; }
        *pre = p;
        InThread(p->rchild, pre);
    }
}
```

## ❗【考点】二叉搜索树（BST）的插入与查找

```c
// BST查找
BiTree BST_Search(BiTree T, KeyType key) {
    while (T && key != T->data) {
        if (key < T->data) T = T->lchild;
        else T = T->rchild;
    }
    return T;
}

// BST插入
int BST_Insert(BiTree *T, KeyType key) {
    if (!*T) {
        *T = (BiTree)malloc(sizeof(BiTNode));
        (*T)->data = key;
        (*T)->lchild = (*T)->rchild = NULL;
        return 1;
    }
    if (key == (*T)->data) return 0;  // 已存在
    if (key < (*T)->data) return BST_Insert(&(*T)->lchild, key);
    return BST_Insert(&(*T)->rchild, key);
}
```

## ❗【考点】图的遍历

### 深度优先搜索（DFS）

```c
int visited[MAXV];
void DFS(Graph G, int v) {
    visit(v);
    visited[v] = 1;
    for (int w = FirstNeighbor(G, v); w >= 0; w = NextNeighbor(G, v, w)) {
        if (!visited[w]) DFS(G, w);
    }
}
```

### 广度优先搜索（BFS）

```c
void BFS(Graph G, int v) {
    Queue Q;
    InitQueue(Q);
    visit(v);
    visited[v] = 1;
    EnQueue(Q, v);
    while (!IsEmpty(Q)) {
        DeQueue(Q, v);
        for (int w = FirstNeighbor(G, v); w >= 0; w = NextNeighbor(G, v, w)) {
            if (!visited[w]) {
                visit(w);
                visited[w] = 1;
                EnQueue(Q, w);
            }
        }
    }
}
```

## ❗【考点】排序算法代码实现

### 快速排序

```c
int Partition(int A[], int low, int high) {
    int pivot = A[low];
    while (low < high) {
        while (low < high && A[high] >= pivot) high--;
        A[low] = A[high];
        while (low < high && A[low] <= pivot) low++;
        A[high] = A[low];
    }
    A[low] = pivot;
    return low;
}

void QuickSort(int A[], int low, int high) {
    if (low < high) {
        int pivotpos = Partition(A, low, high);
        QuickSort(A, low, pivotpos - 1);
        QuickSort(A, pivotpos + 1, high);
    }
}
```

### 堆排序

```c
void AdjustDown(int A[], int k, int len) {
    A[0] = A[k];
    for (int i = 2 * k; i <= len; i *= 2) {
        if (i < len && A[i] < A[i + 1]) i++;
        if (A[0] >= A[i]) break;
        A[k] = A[i];
        k = i;
    }
    A[k] = A[0];
}

void HeapSort(int A[], int len) {
    // 建堆
    for (int i = len / 2; i > 0; i--)
        AdjustDown(A, i, len);
    // 排序
    for (int i = len; i > 1; i--) {
        int temp = A[1]; A[1] = A[i]; A[i] = temp;
        AdjustDown(A, 1, i - 1);
    }
}
```

### 归并排序

```c
void Merge(int A[], int low, int mid, int high) {
    int B[high - low + 1];
    int i = low, j = mid + 1, k = 0;
    while (i <= mid && j <= high) {
        B[k++] = A[i] <= A[j] ? A[i++] : A[j++];
    }
    while (i <= mid) B[k++] = A[i++];
    while (j <= high) B[k++] = A[j++];
    for (k = 0, i = low; i <= high; k++, i++) A[i] = B[k];
}

void MergeSort(int A[], int low, int high) {
    if (low < high) {
        int mid = (low + high) / 2;
        MergeSort(A, low, mid);
        MergeSort(A, mid + 1, high);
        Merge(A, low, mid, high);
    }
}
```
