# 408考研常考题型汇总

## ❗【考点】指令流水线常考题

### 常考题型

1. **流水线加速比与效率计算**：S = T顺序 / T流水，E = n个任务占用的时空区 / n个任务总时空区
2. **数据冲突/控制冲突/结构冲突识别**：
   - 数据冲突（Data Hazard）：后续指令需要前序指令的结果（RAW/WAR/WAW）
   - 控制冲突（Control Hazard）：分支指令改变执行顺序
   - 结构冲突（Structural Hazard）：硬件资源争用
3. **分支预测方法**：静态预测（总是不跳转/总是跳转）、动态预测（BHT分支历史表）
4. **流水线调度**：指令序列中插入气泡（stall）或使用转发（forwarding）消除冲突
5. **超标量与超流水线对比**：超标量并行发射多条指令，超流水线细分流水级

### 典型计算题

**例1**：5段流水线（IF/ID/EX/MEM/WB），每段1个时钟周期，执行100条指令。
- 无冲突：T = 5 + (100-1)×1 = 104 个时钟周期
- 加速比 S = 100×5 / 104 ≈ 4.81

**例2**：判断数据冲突类型
```
I1: ADD R1, R2, R3    ; R1 = R2 + R3
I2: SUB R4, R1, R5    ; 需要 I1 的 R1 → RAW 冲突
```

## ❗【考点】IP子网划分常考题

### 常考题型

1. **给定IP和掩码求网络号/广播地址/主机数**
   - 网络号 = IP AND 子网掩码
   - 广播地址 = 网络号 OR (掩码取反)
   - 主机数 = 2^(32-掩码位数) - 2

2. **CIDR地址块划分**：将一个地址块划分为多个子网
   - 例：192.168.1.0/24 划分为4个子网 → /26，每个子网64个地址

3. **变长子网掩码VLSM**：不同子网使用不同掩码长度
   - 先满足最大子网需求，再逐步细分

4. **路由聚合/超网构造**：将多个连续网络合并为一个
   - 例：192.168.0.0/24 + 192.168.1.0/24 → 192.168.0.0/23

5. **子网数与主机数计算**
   - 借用n位做子网号：子网数 = 2^n
   - 剩余m位做主机号：主机数 = 2^m - 2

### 典型计算题

**例1**：IP = 192.168.1.130，掩码 = 255.255.255.192 (/26)
- 网络号：192.168.1.128
- 广播地址：192.168.1.191
- 主机数：2^6 - 2 = 62

**例2**：将 10.0.0.0/8 划分为至少6个子网
- 需要借用3位（2^3=8≥6），新掩码 /11
- 子网：10.0.0.0/11, 10.32.0.0/11, 10.64.0.0/11, ...

## ❗【考点】进程管理常考题

1. **PV操作应用**：生产者-消费者、读者-写者、哲学家进餐
2. **死锁判断**：资源分配图化简、安全性序列
3. **银行家算法**：安全性检查、资源请求判定
4. **调度算法计算**：周转时间、带权周转时间、响应比

## ❗【考点】页面置换常考题

1. **FIFO/LRU/OPT对比**：给定页面引用串，计算缺页次数
2. **Belady异常**：FIFO增加物理块反而缺页增多
3. **抖动（Thrashing）**：进程频繁缺页，CPU利用率骤降

## ❗【考点】TCP拥塞控制常考题

1. **慢开始/拥塞避免**：cwnd指数增长到ssthresh后线性增长
2. **快重传/快恢复**：收到3个重复ACK后立即重传，cwnd = ssthresh/2
3. **拥塞窗口变化轨迹**：画出cwnd随RTT变化的曲线
