# 408考研学习路线与重点章节

## ❗【考点】操作系统学习路线

### 学习路径

1. **进程管理**：进程概念、调度算法（FCFS/SJF/RR/优先级）、进程同步（信号量/管程）、死锁（条件/银行家算法）
2. **内存管理**：连续分配、分页、分段、虚拟存储器（缺页中断/页面置换）
3. **文件管理**：文件系统、目录结构、磁盘调度（FCFS/SSTF/SCAN/C-SCAN）
4. **I/O管理**：I/O系统、缓冲技术、SPOOLing技术

### 重点章节排序

| 优先级 | 章节 | 核心考点 |
|--------|------|----------|
| ★★★ | 进程管理 | PV操作、死锁、调度算法 |
| ★★★ | 内存管理 | 分页/分段、虚拟存储器、页面置换 |
| ★★ | 文件管理 | 磁盘调度、目录 |
| ★★ | I/O管理 | 缓冲、SPOOLing |

### 学习建议

每部分先理解原理再做计算题。进程管理是重中之重，PV操作和死锁是必考大题。内存管理重点在分页机制和页面置换算法。文件管理重点在磁盘调度计算。

## ❗【考点】数据结构学习路线

### 学习路线

1. **线性表**：顺序表、链表（单链表/双链表/循环链表）
2. **栈和队列**：顺序栈/链栈、顺序队列/链队列、循环队列
3. **树与二叉树**：遍历（前/中/后/层序）、BST、AVL、哈夫曼树、线索化
4. **图**：存储（邻接矩阵/邻接表）、遍历（DFS/BFS）、最短路径（Dijkstra/Floyd）、最小生成树（Prim/Kruskal）、拓扑排序
5. **查找**：顺序查找、折半查找、B树/B+树、哈希表
6. **排序**：插入排序、快排、归并排序、堆排序、基数排序

### 重点章节排序

| 优先级 | 章节 | 核心考点 |
|--------|------|----------|
| ★★★ | 树与二叉树 | 遍历、BST/AVL、哈夫曼 |
| ★★★ | 图 | 最短路径、最小生成树、拓扑排序 |
| ★★★ | 排序 | 快排、堆排、归并、稳定性/复杂度 |
| ★★ | 查找 | 哈希表、B树 |
| ★★ | 线性表 | 链表操作 |

### 学习建议

树和图是算法大题的重点，必须掌握代码实现。排序重点在比较各算法的时间/空间复杂度和稳定性。哈希表是查找的核心。

## ❗【考点】计算机网络学习路线

### 复习路线

1. **物理层**：奈奎斯特定理、香农定理、编码方式
2. **数据链路层**：差错控制（CRC）、流量控制、CSMA/CD、以太网
3. **网络层**：IP地址/子网划分、CIDR、ICMP、路由协议（RIP/OSPF/BGP）
4. **传输层**：TCP/UDP、TCP三次握手/四次挥手、拥塞控制、流量控制
5. **应用层**：DNS、HTTP、FTP、电子邮件

### 重点章节排序

| 优先级 | 章节 | 核心考点 |
|--------|------|----------|
| ★★★ | 传输层 | TCP连接管理、拥塞控制 |
| ★★★ | 网络层 | IP子网划分、路由协议 |
| ★★ | 数据链路层 | CSMA/CD、CRC |
| ★★ | 物理层 | 奈奎斯特/香农 |

### 学习建议

重点在TCP和IP。传输层的TCP三次握手/四次挥手、拥塞控制是必考。网络层的子网划分计算是常考题型。每层重点掌握协议功能和数据单元。

## ❗【考点】计算机组成原理重点章节

### 重点章节

1. **数据表示**：补码、浮点数IEEE754、定点/浮点运算
2. **运算器**：加法器、ALU功能
3. **存储系统**：Cache（映射方式/替换算法/写策略）、主存、虚拟存储器
4. **指令系统**：寻址方式、CISC/RISC
5. **CPU**：指令流水线（数据冲突/控制冲突/结构冲突）、控制器（硬布线/微程序）
6. **总线**：总线仲裁、总线宽度
7. **I/O系统**：中断、DMA、I/O方式

### 重点章节排序

| 优先级 | 章节 | 核心考点 |
|--------|------|----------|
| ★★★ | 存储系统 | Cache映射、虚拟存储器 |
| ★★★ | CPU | 流水线、控制器 |
| ★★★ | 数据表示 | IEEE754、补码运算 |
| ★★ | 指令系统 | 寻址方式 |
| ★★ | I/O系统 | DMA、中断 |

### 学习建议

存储系统是重中之重，Cache映射和虚拟存储器是必考大题。CPU流水线重点在冲突检测和性能计算。数据表示重点在浮点数格式和运算。
